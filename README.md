# Telegram Fact-Checking Bot

Асинхронный Telegram-бот для проверки новостей на достоверность с интеграцией живого веб-поиска. Работает на базе моделей Groq (Llama 3.3 70B) и подтягивает актуальные данные из сети в реальном времени.

## Особенности
- **Живой веб-поиск**: Бот сам ищет информацию в сети по запросу пользователя перед анализом.
- **Актуальные даты**: ИИ всегда знает текущую дату благодаря динамическому системному промпту.
- **Легковесность**: Поиск реализован без тяжелых библиотек, не нагружает оперативную память сервера.
- **Бесплатные лимиты**: Использование Groq API позволяет обрабатывать до 14 400 запросов в сутки бесплатно.

## Стек технологий
- Python 3.13
- aiogram 3.13.1
- groq
- curl_cffi & BeautifulSoup4

## Установка и запуск

1. Склонируйте репозиторий:
git clone https://github.com/Arslan52-best/Factchecker_bot.git
cd Factchecker_bot

2. Установите необходимые зависимости:
pip install -r requirements.txt

3. Откройте файл config.py и укажите ваши ключи:
- BOT_TOKEN — токен вашего Telegram-бота от BotFather.
- GROQ_API_KEY — API ключ от панели Groq Cloud.

4. Запустите бота:
python app.py
